## Purpose

Compute the common base 10 logarithm.

## Description

*Usage: REAL ALOG10, X, Y Y = ALOG10(X) *Arguments: X :IN This is the argument. It will not be modified by ALOG. *Function Return Values: ALOG10 : the base 10, i.e. common, logarithm. *Description: ALOG10 evaluates the base 10 logarithm of an argument. The argument X must be positive. For a non-positive argument, an error message is generated by XERMSG and the routine aborts.
