## Purpose

Compute the double precision arccosine.

## Description

*Usage: DOUBLE PRECISION DACOS, X, Y Y = DACOS(X) *Arguments: X :IN This is the argument. It will not be modified by DACOS. *Function Return Values: DACOS : the double precision arccosine of X. *Description: DACOS evaluates the double precision arccosine of a double precision argument. The argument X must be in the interval -1.0 .LE. X .LE. +1.0. For arguments outside the permitted interval, an error message is generated by XERMSG and the routine aborts.
