## Purpose

Subsidiary to BSKIN

## Description

BDIFF computes the sum of B(L,K)*V(K)*(-1)**K where B(L,K) are the binomial coefficients. Truncated sums are computed by setting last part of the V vector to zero. On return, the binomial sum is in V(L).
