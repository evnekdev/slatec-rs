## Purpose

Compute the argument of a complex number.

## Description

CARG(Z) calculates the argument of the complex number Z. Note that CARG returns a real result. If Z = X+iY, then CARG is ATAN(Y/X), except when both X and Y are zero, in which case the result will be zero.
