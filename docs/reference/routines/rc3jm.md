## Purpose

Evaluate the 3j symbol g(M2) = (L1 L2 L3 ) (M1 M2 -M1-M2) for all allowed values of M2, the other parameters being held fixed.

## Description

Usage: REAL L1, L2, L3, M1, M2MIN, M2MAX, THRCOF(NDIM) INTEGER NDIM, IER CALL RC3JM (L1, L2, L3, M1, M2MIN, M2MAX, THRCOF, NDIM, IER) Although conventionally the parameters of the vector addition coefficients satisfy certain restrictions, such as being integers or integers plus 1/2, the restrictions imposed on input to this subroutine are somewhat weaker. See, for example, Section 27.9 of Abramowitz and Stegun or Appendix C of Volume II of A. Messiah. The restrictions imposed by this subroutine are 1. L1.GE.ABS(M1) and L1+ABS(M1) must be an integer; 2. ABS(L1-L2).LE.L3.LE.L1+L2;
