## Purpose

Subsidiary to SPLP

## Description

Given a "virtual" location, IPLOC returns the relative working address of the vector component stored in SX, IX. Any necessary page swaps are performed automatically for the user in this function subprogram. LOC is the "virtual" address of the data to be retrieved. SX ,IX represent the matrix where the data is stored.
