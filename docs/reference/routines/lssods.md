## Purpose

Subsidiary to BVSUP

## Description

LSSODS solves the same problem as SODS (in fact, it is called by SODS) but is somewhat more flexible in its use. In particular, LSSODS allows for iterative refinement of the solution, makes the transformation and triangular reduction information more accessible, and enables the user to avoid destruction of the original matrix A. Modeled after the ALGOL codes in the articles in the REFERENCES section. **********************************************************************
