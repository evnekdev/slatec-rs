## Purpose

Compute the double precision arctangent in the proper quadrant.

## Description

*Usage: DOUBLE PRECISION DATAN2, SN, CS, Y Y = DATAN2(SN, CS) *Arguments: SN :IN This is an argument. It will not be modified by DATAN2. CS :IN This is an argument. It will not be modified by DATAN2. *Function Return Values: DATAN2 : the double precision arctangent in the proper quadrant. *Description: DATAN2 evaluates the double precision arctangent of two double precison arguments in the proper quadrant. The arguments SN and CS must not both be zero. If both of the two arguments are zero, an error message is generated by XERMSG and the routine aborts.
