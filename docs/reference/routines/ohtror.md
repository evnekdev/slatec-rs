## Purpose

Subsidiary to BVSUP

## Description

For a rank deficient problem, additional orthogonal HOUSEHOLDER transformations are applied to the right side of Q to further reduce the triangular form. Thus, after application of the routines ORTHOL and OHTROR to the original matrix, the result is a nonsingular triangular matrix while the remainder of the matrix has been zeroed out.
