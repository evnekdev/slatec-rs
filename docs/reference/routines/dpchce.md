## Purpose

Set boundary conditions for DPCHIC

## Description

DPCHCE: DPCHIC End Derivative Setter. Called by DPCHIC to set end derivatives as requested by the user. It must be called after interior derivative values have been set. To facilitate two-dimensional applications, includes an increment between successive values of the D-array.
