## Purpose

Subsidiary to ZBESI and ZBESK

## Description

ZBUNI COMPUTES THE I BESSEL FUNCTION FOR LARGE ABS(Z).GT. FNUL AND FNU+N-1.LT.FNUL. THE ORDER IS INCREASED FROM FNU+N-1 GREATER THAN FNUL BY ADDING NUI AND COMPUTING ACCORDING TO THE UNIFORM ASYMPTOTIC EXPANSION FOR I(FNU,Z) ON IFORM=1 AND THE EXPANSION FOR J(FNU,Z) ON IFORM=2
