## Purpose

Compute the double precision arcsine.

## Description

*Usage: DOUBLE PRECISION DASIN, X, Y Y = DASIN(X) *Arguments: X :IN This is the argument. It will not be modified by DASIN. *Function Return Values: DASIN : the double precision arcsine of X. *Description: DASIN evaluates the double precision arcsine of a double precision argument. The value for the argument X must be in the interval -1.0 .LE. X .LE. +1.0. For arguments outside the permitted interval, an error message is generated by XERMSG and the routine aborts.
