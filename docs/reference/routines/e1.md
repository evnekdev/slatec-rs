## Purpose

Compute the exponential integral E1(X).

## Description

E1 calculates the single precision exponential integral, E1(X), for positive single precision argument X and the Cauchy principal value for negative X. If principal values are used everywhere, then, for all X, E1(X) = -Ei(-X) or Ei(X) = -E1(-X).
