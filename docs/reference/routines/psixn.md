Subsidiary to EXINT

## Description

This subroutine returns values of PSI(X)=derivative of log GAMMA(X), X .GT. 0.0 at integer arguments. A table look-up is performed for N .LE. 100, and the asymptotic expansion is evaluated for N .GT. 100.
